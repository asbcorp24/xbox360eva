# Xbox360 EVA

Стартовый проект одиночного 3D-рогалика для Xbox 360 RGH/JTAG.

Проект строится по двухконтурной схеме:

- `platform/desktop` — быстрый запуск и проверка игровой логики на ПК;
- `platform/xenon` — платформенный слой для Xbox 360 через LibXenon.

На текущем этапе реализовано ядро прототипа: процедурная генерация комнат, игрок, противники, бой, лут, переход между этажами и текстовый desktop-режим для проверки логики. Графический рендерер LibXenon и чтение геймпада вынесены в отдельный слой и будут подключаться без переписывания игровой механики.

## Концепция

Игрок проходит случайно созданные кубические подземелья. Каждый этаж содержит стартовую комнату, боевые комнаты, сокровищницу, элитную комнату и босса. После смерти забег начинается заново.

## Быстрый запуск на ПК

Требования:

- CMake 3.16 или новее;
- компилятор с поддержкой C++17.

```bash
cmake -S . -B build
cmake --build build
./build/xbox360eva
```

Windows PowerShell:

```powershell
cmake -S . -B build
cmake --build build --config Release
.\build\Release\xbox360eva.exe
```

## Управление desktop-прототипом

- `w`, `a`, `s`, `d` — переход между соседними комнатами;
- `x` — атака;
- `h` — использовать аптечку;
- `n` — новый забег;
- `q` — выход.

## Структура

```text
src/core       игровая логика без зависимости от платформы
src/game       состояние забега и основной цикл
platform       desktop и Xbox 360 адаптеры
```

## Xbox 360

Для сборки требуется открытый toolchain LibXenon. Проприетарный Microsoft XDK в репозиторий не входит.

План Xbox-слоя:

1. Инициализация видео LibXenon.
2. Чтение Xbox 360 Controller.
3. Отрисовка кубической геометрии комнат.
4. Камера от третьего лица.
5. HUD, модели врагов и предметов.
6. Сборка исполняемого файла для запуска через XeXMenu/Aurora.

## Статус

Это первый рабочий каркас, а не законченная игра. Он предназначен для последовательной разработки и тестирования механик до подключения полноценного 3D-рендера на консоли.
