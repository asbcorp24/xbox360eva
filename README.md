# EVA: Розовое Королевство

Добрый одиночный 3D-рогалик для **Евы Балабановой** и Xbox 360 RGH/JTAG.

Ева путешествует по случайно создаваемым конфетным островам, собирает потерянные звёздочки, освобождает котят и побеждает не страшных врагов с помощью волшебной палочки, сердечек и радужной магии.

## Стиль игры

- всё розовое, яркое и милое;
- кубическая low-poly графика;
- розовые замки, облака, леденцы и цветы;
- котики, единороги и добрые слизняшки;
- никаких крови, жестокости и пугающих сцен;
- каждый новый забег создаёт другую карту;
- после прохождения острова открываются новые наряды и украшения.

## Главная героиня

**Ева Балабанова** — принцесса-хранительница Розового Королевства.

Начальные способности:

- волшебная палочка;
- выстрел сердечком;
- радужный рывок;
- защитный пузырь;
- призыв котёнка-помощника.

## Комнаты уровня

- Розовая поляна — старт;
- Конфетная дорожка — обычная комната;
- Кошачье кафе — восстановление здоровья;
- Комната подарков — случайный предмет;
- Радужный сад — испытание;
- Дворец Облачной Королевы — босс этажа.

## Первый прототип

В репозитории находится переносимое C++17-ядро игры:

- генерация случайной карты;
- комнаты разных типов;
- характеристики Евы;
- звёздочки, сердечки и подарки;
- добрые противники;
- переход между этажами;
- текстовый режим для проверки механики на ПК;
- заготовка платформенного слоя для LibXenon.

## Сборка на ПК

```bash
cmake -S . -B build
cmake --build build
./build/eva_pink_kingdom
```

Windows:

```powershell
cmake -S . -B build
cmake --build build --config Release
.\build\Release\eva_pink_kingdom.exe
```

## Управление прототипом

- `w`, `a`, `s`, `d` — перейти в соседнюю комнату;
- `x` — использовать волшебную палочку;
- `h` — съесть волшебный кекс и восстановить здоровье;
- `n` — начать новое приключение;
- `q` — выход.

## Xbox 360

Целевая платформа — Xbox 360 RGH/JTAG через открытый LibXenon toolchain. Проприетарный Microsoft XDK в проект не входит.

Следующий технический этап: подключение 3D-рендера, камеры от третьего лица и геймпада Xbox 360.
